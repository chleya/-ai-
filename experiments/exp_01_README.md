## 系统定义

```
S = (X, Φ, C, T)

X: 64维连续状态空间
Φ: 线性演化 + 噪声
C: L2归一化约束
T: 100000步离散时间
```

## 设计原理

### 为什么是最小系统？

1. **状态空间足够大**: 64维足以产生复杂行为
2. **演化足够简单**: 线性 + 噪声，没有隐藏复杂性
3. **约束足够强**: 归一化防止状态爆炸

### 假设

1. 能量（x·x）应该收敛到某个值
2. 熵（状态分散度）应该趋于稳定
3. 如果有吸引子，应该能观察到轨迹聚集

### 预期结果

| 场景 | 可能性 | 系统命运 |
|------|-------|---------|
| 固定点吸引子 | 高 | 系统收敛到单一状态 |
| 极限环 | 中 | 系统在低维流形上循环 |
| 混沌 | 低 | 状态持续不可预测 |
| 无吸引子 | 极低 | 系统崩溃 |
