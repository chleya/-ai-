# 噪声诱导的超稳态研究

## 核心问题

为什么50%噪声反而提高存活率？

## 实验设计

### 噪声类型

| 噪声类型 | 分布 | 物理意义 |
|----------|------|----------|
| 高斯噪声 | N(0, σ²) | 热噪声 |
| 脉冲噪声 | δ(t) | 电磁脉冲 |
| 均匀噪声 | U(-a, a) | 环境干扰 |

### 假说

**物理假说**：噪声充当"动态搅拌器"，防止系统固化在大规模系统中。

## 预期结果

| 噪声类型 | 存活率 | 机制 |
|----------|---------|------|
| 无噪声 | 基准 | 自然演化 |
| 低噪声(10%) | 略高 | 正则化 |
| 中噪声(30%) | 更高 | 充分搅拌 |
| 高噪声(50%) | 最高 | 强搅拌 |

## 数学描述

### 噪声驱动的动力学

$$ds = f(s)dt + \sigma dW_t$$

其中 $\sigma$ 是噪声强度。

### Fokker-Planck方程

$$\frac{\partial p}{\partial t} = -\frac{\partial}{\partial s}[f(s)p] + \frac{\sigma^2}{2}\frac{\partial^2}{\partial s^2}[p]$$

**关键洞察**：噪声项导致概率分布扩散，防止系统陷入局部最优。

# 跨任务的元稳定性研究

## 核心问题

任务切换时，哪种模式响应更快？

## 实验设计

### 任务切换协议

```
T=0-400:  目标 P
T=400-800: 目标 P'
```

### 对比指标

| 指标 | 定义 |
|------|------|
| 切换延迟 | 从P切换到P'后达成新共识的时间 |
| 精度 | 切换后达成正确共识的比例 |
| 抖动 | 切换过程中的方差 |

## 假说

**核心假说**：Rand组在任务切换时具有压倒性优势，因为没有历史包袱。

## 预期结果

| 指标 | Peak组 | Rand组 | 优势方 |
|------|---------|---------|--------|
| 切换延迟 | 高 | 低 | Rand |
| 精度 | 低 | 高 | Rand |
| 抖动 | 高 | 低 | Rand |

## 数学描述

### 历史包袱模型

$$W_{new} = W_{old} + \alpha(P' - P_{memory})$$

其中 $P_{memory}$ 是旧任务的记忆。

### 无偏切换

$$W_{new} = W_{random} + \alpha P'$$
