## 三、蝉蜕协议设计

### 3.1 核心思想

```
单体系统（死亡） → 种群生态（永续）
     │                   │
     ▼                   ▼
  长治久安           周期性新生
     │                   │
     ▼                   ▼
  必然崩溃           死亡→重生
```

### 3.2 L6演化层：适者生存

**设计**：10个子系统并行演化

```python
# 初始化种群
populations = []
for i in range(10):
    W = random_weights(N=60)
    populations.append({
        'W': W,
        'alive': True,
        'knowledge': None
    })
```

**机制**：
- 每200步检查每个子系统
- 淘汰死亡子系统（共识<50%）
- 用存活子系统覆盖死亡子系统

### 3.3 L3记忆层：知识继承

**设计**：保存每个子系统的"峰值状态"

```python
# 保存峰值知识
peak_state = None
for e in range(200):
    s = evolve()
    if is_consensus(s):
        if peak_state is None or quality(s) > quality(peak_state):
            peak_state = s.copy()

# 继承给下一代
if peak_state is not None:
    P = peak_state  # 用峰值初始化
```

**作用**：
- 死亡子系统重生时使用上一轮的峰值
- 避免从头学习
- 实现知识累积

### 3.4 L5群体层：多次演化

**设计**：3轮脉冲演化

```python
for round_idx in range(3):
    for sub_idx in range(10):
        if not populations[sub_idx]['alive']:
            # 复活，使用知识继承
            populations[sub_idx]['W'] = populations[alive_subsys[0]]['W'].copy()
        
        evolve(sub_idx, steps=200)
```
