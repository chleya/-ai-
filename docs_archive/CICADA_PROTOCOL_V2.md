## 范式突破

### 核心发现

> **"知识继承是蝉蜕协议成功的关键，单次增益+27%。"**

### 机制原理

```
第1轮                 第2轮                 第3轮
  │                     │                     │
  ▼                     ▼                     ▼
死亡→重生            死亡→重生            死亡→重生
  │                     │                     │
  ▼                     ▼                     ▼
学习P1              学习P2              学习P3
  │                     │                     │
  └─────────────────────┴─────────────────────┘
                        │
                        ▼
               累积知识：P1+P2+P3
                        │
                        ▼
               越往后越快达成共识
```

### 数学描述

$$P_{n+1} = P_n + \alpha \cdot \text{Peak}_n$$

其中：
- $P_n$：第n轮的先验知识
- $\text{Peak}_n$：第n轮的峰值状态
- $\alpha$：继承率

## 理论框架

### 1. 知识继承的物理图像

**之前**：
- 系统死亡后重生
- 重新从随机状态开始
- 每次都是"从零开始"

**现在**：
- 系统死亡后重生
- 继承上一轮的峰值知识
- 每次都是"站在巨人的肩膀上"

### 2. 知识编码

**峰值状态向量**：

```python
peak_state = None
for e in range(200):
    s = evolve()
    if is_consensus(s):
        if peak_state is None or quality(s) > quality(peak_state):
            peak_state = s.copy()
```

**作用**：
- 编码了"共识是如何达成的"
- 包含了"正确的方向信息"
- 是系统的"经验结晶"

### 3. 知识继承

**初始化**：

```python
if last_peak is not None:
    P = last_peak.copy()  # 用峰值初始化
else:
    P = random()  # 随机初始化
```

**效果**：
- 起点更高
