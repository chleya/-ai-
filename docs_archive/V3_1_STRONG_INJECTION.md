## 一、间歇性混沌的本质

### 1.1 观察到的现象

| 现象 | 含义 |
|------|------|
| 0% ↔ 100% 跳变 | 系统在多个吸引子间跳跃 |
| 快照无效 | 快照记录的是"位置"而非"引力结构" |
| 无平滑衰减 | 不是渐进遗忘，而是突然跳跃 |

### 1.2 数学描述

**非线性动力学中的间歇性**：

$$\frac{ds}{dt} = F(s) + \xi(t)$$

其中 $\xi(t)$ 是噪声，$F(s)$ 是非线性动力学。

**跳变条件**：

当 $||\nabla V(s)|| < ||\xi(t)||$ 时，系统会从一个吸引子跳到另一个。

### 1.3 吸引子图景

```
           A (真理)                    C (伪真理)
           深坑                        浅坑
             \                        /
              \    噪声跳跃        /
               \______/\_________/
                      /
              混沌边界区域

- 300步时：虽然在A中，但坑已经开始变浅
- 500步后：坑太浅，噪声可以轻松跳出
- 1100步：偶然跳回A，但很快又跳出
```

## 二、为什么300步的快照是"幻觉"？

### 2.1 两种快照

| 类型 | 内容 | 价值 |
|------|------|------|
| **位置快照** | 记录小球在A中心 | ❌ 无用 |
| **结构快照** | 记录重力结构 | ✅ 有效 |

### 2.2 300步快照的问题

```
时刻 | 小球位置 | Basin深度 | 快照价值
-----|---------|-----------|----------
100步 | A中心 | 深 | ✅ 有效
300步 | A中心 | 浅 | ⚠️ 幻觉
500步 | A/C边界 | 极浅 | ❌ 无效
```

**关键洞察**：

> "300步的100%只是'看起来在A中心'，但此时Basin已经浅到无法束缚小球的微小扰动。"

## 三、v3.1 强力注入设计

### 3.1 核心思路

**之前**：
- 快照时机：300步（太晚）
- 快照占比：0.7（太温柔）
- 结果：记录了"浅坑"

**v3.1**：
- 快照时机：150步（更早）
- 快照占比：0.95-1.0（更激进）
- 目标：记录"深坑"

### 3.2 参数设计

| 参数 | 值 | 理由 |
|------|-----|------|
| 快照时机 | 150步 | LTD已清除噪声，Basin还深 |
| 快照占比 | 0.95-1.0 | 强力注入"深坑" |
| 恢复时机 | 500步 | 衰减刚开始时介入 |

### 3.3 物理图像

```
150步：黄金时刻
         │
         ▼ 快照
      ┌─────────────┐
      │  深坑快照   │ ← 保存真正的引力结构
      └─────────────┘
         │
         ▼ 演化
      ┌─────────────┐
      │  坑变浅     │ ← 衰减开始
      └─────────────┘
         │
         ▼ 500步
      ┌─────────────┐
      │ 强力注入     │ ← 0.95-1.0比例
      └─────────────┘
         │
         ▼ 预期结果
      ┌─────────────┐
      │  坑恢复深   │ ← Basin被强制还原
      └─────────────┘
```

## 四、预期 vs 现实

### 4.1 如果成功

| 指标 | 预期 |
|------|------|
| 150步快照 | 捕捉真正的"深坑" |
| 500步注入 | Basin被强制还原 |
| 长期演化 | 衰减被抑制 |
| 最终正确率 | >65% 基线 |

### 4.2 如果失败

**颠覆性结论**：

> "系统的不稳定性不是源于权重的丢失，而是源于演化轨迹本身具有'不可逆的耗散性'。"

这意味着：
1. 没有任何快照能捕捉"真正的引力结构"
2. Basin的坍缩从一开始就发生了
3. 任何"恢复"都只是在延缓必然的结局
