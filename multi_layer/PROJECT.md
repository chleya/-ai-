# Multi-Layer Cascade: Scaling Through Communication

**基于**: System Stability 项目 (FINAL_REPORT.md)  
**日期**: 2026-02-07  
**起点**: N=50 单层系统

---

## 研究背景

### 核心问题

System Stability项目证明了N=50的Hebbian系统具备联想记忆能力：

| 能力 | 验证 |
|------|------|
| 双稳态 | ✓ (+/-1) |
| 模式补全 | ✓ (10%→100%) |
| 抗噪 | ✓ (σ=10→80%) |
| 干扰极限 | 71% |

**下一个问题**: 如何扩展到更大规模？

### 生物学启示

大脑皮层是多层次的：
- 初级皮层处理简单特征
- 高级皮层整合复杂信息
- 层级间存在双向通信

### 假设

> 两个N=50单元通过双向通信可以产生更复杂的行为，同时保持各自的记忆能力。

---

## 研究问题

### 主问题

多层级系统能否比单层系统具备更强的记忆和泛化能力？

### 子问题

1. **通信模式**: 单向vs双向，哪个更有效？
2. **干扰传递**: 上层干扰是否影响下层？
3. **特征整合**: 能否从多个模式中提取共同特征？
4. **扩展性**: 这种结构能否扩展到更多层？

---

## 系统设计

### 架构

```
Layer 1 (N=50) ←→ Layer 2 (N=50)

通信: x2 = f(W21 · x1)
       x1 = f(W12 · x2)
```

### 变体

| 变体 | 描述 |
|------|------|
| 单向 | L1 → L2 |
| 双向 | L1 ↔ L2 |
| 循环 | L1 ↔ L2 ↔ L1 ... |
| 层级 | L1 → L2 → L3 |

### 学习规则

**各层独立Hebbian**:

$$W_{11} += \eta \cdot x_1 x_1^T$$
$$W_{22} += \eta \cdot x_2 x_2^T$$

**跨层连接**:

$$W_{12} = \text{learned from data}$$
$$W_{21} = \text{learned from data}$$

### 基线（单层）

| 指标 | 值 |
|------|------|
| 干扰率 | 71% |
| 补全率 | 100% |
| 抗噪 | 80% |

### 实验1: 双层通信

```
1. 创建2个N=50单元
2. 训练L1存储模式P1
3. 训练L2存储模式P2
4. 建立跨层连接
5. 测试:
   - 单独访问P1
   - 单独访问P2
   - 联合访问 (L1↔L2)
```

### 实验2: 特征整合

```
1. 创建L1存储模式A
2. 创建L2存储模式B
3. 模式A和B有共同特征C
4. 测试: 能否从C提取A+B的整合信息？
```

### 实验3: 干扰传递

```
1. L1存储P1, P2 (高干扰)
2. L2存储Q1, Q2
3. 测试:
   - L1干扰是否传递到L2？
   - L2能否隔离L1的干扰？
```

## 关键假设检验

### H1: 双向优于单向

$$H_0: \text{双向性能} = \text{单向性能}$$
$$H_1: \text{双向性能} > \text{单向性能}$$

### H2: 层级减少干扰

$$H_0: \text{双层干扰率} = \text{单层干扰率}$$
$$H_1: \text{双层干扰率} < \text{单层干扰率}$$

### H3: 特征可以整合

$$H_0: \text{整合成功率} = \text{随机}$$
$$H_1: \text{整合成功率} > \text{随机}$$
