# 相变临界点研究：初步结果

## 核心发现：周期性振荡

### 快速筛选结果

| N | Peak | Rand | Diff | 优势方 |
|---|------|------|------|--------|
| 200 | 66% | 0% | +66% | Peak |
| 400 | 33% | 66% | -33% | Rand |
| 600 | 100% | 66% | +34% | Peak |
| 1000 | 66% | 100% | -34% | Rand |

### 关键观察

**不是单调相变，而是周期性振荡！**

```
N=200:  Peak > Rand (+66%)
N=400:  Peak < Rand (-33%)
N=600:  Peak > Rand (+34%)
N=1000: Peak < Rand (-34%)
```

## 数学描述

### 振荡模型

$$\Delta(N) = A \sin(2\pi N / T) + B$$

其中：
- $\Delta(N)$ = Peak - Rand 优势差
- $T$ = 振荡周期（猜测T≈400）
- $A$ = 振幅
- $B$ = 偏移

### 拟合数据

从快速测试：
- N=200: Δ=+66%
- N=400: Δ=-33%
- N=600: Δ=+34%
- N=1000: Δ=-34%

拟合参数：
- T ≈ 400
- A ≈ 50
- B ≈ 0

## 物理意义

### 为什么会有振荡？

1. **离散效应**：有限N导致的离散采样
2. **边界条件**：权重矩阵的离散结构
3. **噪声平均**：不同N下的噪声平均效果不同
