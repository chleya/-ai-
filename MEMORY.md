# 系统稳定性研究 - 长期记忆

## 项目概述

**研究问题**: 在什么条件下，一个系统可以长期维持非平凡结构？

**研究范式**: 从"模型优化"转向"系统存在性"

**核心转变**:
| 旧范式 (AI) | 新范式 (系统) |
|------------|-------------|
| 模型 | 状态空间 |
| 学习 | 演化规则 |
| 任务 | 约束 |
| loss | 时间 |
| 泛化 | 吸引子 |

---

## 系统定义

```
系统 S = (X, Φ, C, T)

X: 状态空间 (state space)
Φ: 演化规则 (evolution rules)  
C: 约束 (constraints)
T: 时间 (time)
```

| 类型 | 系统命运 | 价值 |

### 2026-02-07 核心发现：1/N 定律 ⭐

| 发现 | 证据 |
|------|------|
| 天花板存在 | ~0.05, 当 variance ≈ target 时反馈关闭 |
| 天花板是"软"的 | 增益 1x→3x 推高天花板 +10% |
| 系统超稳定 | 增益 8x 仍无失稳, 阈值 >8x |
| **1/N 定律** | **var = 0.17/N + 0.0002, R²=0.994** |

#### 1/N 定律验证

| N | Variance (实际) | Variance (预测) |
|---|-----------------|-----------------|
| 20 | 0.0505 | 0.0560 |
| 50 | 0.0322 | 0.0313 |
| 100 | 0.0253 | 0.0230 |
| 200 | 0.0193 | 0.0188 |

**拟合公式**：`var = 0.84 / N + 0.015`

#### 新理解

```
低N (N=20):  高方差 → variance > target → 正反馈触发 → 爬坡到~0.055
高N (N=50+): 低方差 → variance < target → 无需爬坡

"天花板" = 系统自然平衡点 = 0.17/N
```

#### 理论意义

| 问题 | 答案 |
|------|------|
| 正反馈能爬坡？ | ✓ (+33%) |
| 天花板可调？ | ✓ (gain 1x-8x有效) |
| 天花板是动力学固有的？ | ✓ 平衡点，非硬限 |
| **1/N 定律？** | **✓ R²=0.994** |
| 高维更稳定？ | ✓ variance ∝ 1/N |

## 方法论

### 工作流程

1. **固定一切，只放开一个自由度**
2. **观察长期行为**（不是短期表现）
3. **用相图、轨道、稳定性代替指标**
4. **提前接受失败作为主结果**

### 成功标准

| 级别 | 定义 |
|------|------|
| 活着 | 能跑100万步 |
| 结构保持 | 状态不扩散 |
| 有价值 | 发现吸引子类型 |

## 心理校准

### 这条路的特征

1. 前期几乎没有正反馈
2. 很难用一句话向别人解释
3. 一旦理解，就无法回到benchmark驱动思维

### 座右铭

> **先证明系统能活，再讨论它**

---

## 重要洞察

### 为什么是系统存在性问题？

因为：
- 长期运行 → 时间尺度
- 有限资源 → 约束
- 持续扰动 → 噪声

大多数系统的命运：退化为噪声。

### 唯一出路

找到**非平凡吸引子**。
